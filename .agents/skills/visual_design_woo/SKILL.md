# Visual design — sklep Woo (narzędzia / B2B)

## Cel

Strona ma wyglądać jak **profesjonalny sklep** (frezy, narzędzia), a nie „szablon z bazaru”. Klient ma zobaczyć: porządek, czytelność, zaufanie.

## Zasady

- **Spójność:** jeden zestaw kolorów (główny, akcent, tło), jedna–dwie czcionki. Nagłówki i ciało tekstu rozróżnialne.
- **Białe przestrzenie:** nie zagęszczać; odstępy między blokami ułatwiają skanowanie.
- **Hierarchia:** najważniejsze (cena, CTA, dostępność) — większe / wyraźniejsze; drugoplanowe (tagi, kategorie) — mniejsze.
- **Przyciski:** jeden styl główny (np. „Dodaj do koszyka”), drugi styl dla linków i akcji pobocznych.
- **Zaufanie:** ikony dostawy, płatności, „Faktura VAT”, dostępność — w widocznym miejscu przy CTA lub w stopce.

## Strona główna (z notatek klienta)

- Logo + wyszukiwarka + konto na górze.
- Menu: Sklep, O nas, Pomoc, Kontakt.
- Baner „dlaczego my” (bez nadmiaru przycisków).
- Blok z kategoriami towarów.
- Karuzela / losowe produkty („wylosowane” do wyboru).
- Blok z systemem rabatów (lojalność: 1000 / 3000 / 5000 / 10000 zł → 5–20%).
- Instagram — jeden rząd postów, link do profilu.
- Stopka: kontakt, godziny, link „Sklep” (bez listy kategorii).

## Odniesienia

- Staging.kayer.pl — układ głównej (bez kopiowania; inspiracja struktury).
- Allegro — czytelne karty produktu, oceny, „w magazynie” przy cenie.
