# Правила каталога

## Атрибуты (WC attributes) минимум
- Материал обработки
- Тип операции
- Диаметр
- Длина
- Хвостовик/цанга
- Покрытие
- Кол-во зубьев

## Карточка товара (структура)
1) 1–2 строки: для чего
2) "Подходит для / Не подходит для"
3) Характеристики таблицей
4) Комплектация/совместимость
5) FAQ
6) Доставка/возврат рядом с CTA

## Названия и SKU
- единый формат: TYPE-Ø-L-COAT-FLUTES-SHANK
- без "лишней поэзии", только параметры
