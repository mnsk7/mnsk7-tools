# As-Is Audit WP/Woo

## Быстрые сигналы денег (P0/P1)

- Карточка товара: цена / наличие / доставка / возврат видны без скролла?
- add-to-cart работает быстро?
- Фильтры помогают выбрать по материалу / операции?
- Поиск по SKU / диаметру работает?

## Контент

- Атрибуты заполнены системно или хаос?
- Фото: есть alt? Вес не убивает LCP?

## SEO

- Категории имеют уникальные Title / H1?
- Фильтры не создают индексируемый мусор?

## Tech

- Overrides Woo в теме: где и сколько?
- Кастом код: есть ли mini-plugin / mu-plugin?
- Список плагинов: критичные / лишние
