## Основные документы (краткое содержание)

### Extension Integrations

Storefront совместим со всеми расширениями WooCommerce с WooCommerce.com. Ряд расширений глубоко интегрированы с ядром Storefront (Advanced Product Labels, Ajax Layered Navigation, Composite Products, Product Bundles, Product Gallery Slider, Product Reviews Pro и др.).

### Installing Storefront in your language

Переводы Storefront на WordPress.org. Языковые пакеты: Storefront Language Packs. Загрузка в `wp-content/languages/themes/storefront-{locale}.mo`. Настройка: Settings → Site Language. Кастомные переводы: Loco Translate, PoEdit.

### Remove 'Designed by WooThemes' from footer

Кредиты в футере настраиваются в Customizer, раздел «Footer Customization». Для Storefront и child-тем — Storefront Powerpack или кастомный CSS.

### Storefront Customizer

Внешний вид настраивается через WordPress Customizer. Доступ: Admin bar → Customize или Appearance → Customize. Секция: Storefront Settings.

### Storefront FAQ

- **Логотип:** Storefront 2.0 + WordPress 4.5 — Site Identity в Appearance → Customize (Add, Delete or Change a Custom Logo).
- **Главная страница:** Homepage Control для перестановки и включения/выключения блоков.
- И другие частые вопросы.

### Storefront Installation & Configuration

Официальная бесплатная тема WooCommerce. Установка через админку; после установки — подсказки и guided tour в Customizer. Проверить системные требования (WooCommerce, WordPress).

### Storefront Requirements & Version Support

Требования к серверу — как у WooCommerce core. Storefront может работать без WooCommerce (как обычная тема WP). Политика поддержки версий описана в документе.

### Translating Storefront

Управление переводами на WordPress.org. Установка языка: см. Installing Storefront in Your Language.
