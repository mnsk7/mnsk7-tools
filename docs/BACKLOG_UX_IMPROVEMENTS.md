# Декомпозиция: улучшения UX/UI и мобильная версия

Источник: фидбек по главной, категориям, карточке товара, корзине. Приоритеты и зависимости — для планирования спринтов.

---

## 1. Главная страница

### 1.1 Блок «Bestsellery i polecane» — видимость цены
| ID | Задача | Детали |
|----|--------|--------|
| 1.1.1 | Подписать цену на карточках | Добавить «zł» или «PLN» рядом с числом |
| 1.1.2 | Разместить цену под названием товара | Единое место: название → цена → кнопка |
| 1.1.3 | Выделить цену цветом | Отдельный цвет (например, акцентный) и/или жирный шрифт |

### 1.2 Кнопки на главной — призывы к действию
| ID | Задача | Детали |
|----|--------|--------|
| 1.2.1 | Hover для «Przejdź do sklepu» | Изменение фона/тени/подсветки при наведении |
| 1.2.2 | Hover/active для «Dodaj do koszyka» | То же + состояние :active при клике |
| 1.2.3 | Единый стиль hover для всех CTA | Проверить все кнопки в блоках главной |

### 1.3 Адаптивность главной (мобильные, iPhone XR)
| ID | Задача | Детали |
|----|--------|--------|
| 1.3.1 | Категории: 1–2 колонки на мобильных | Вместо 4 колонок — сетка 1–2 кол. или горизонтальная прокрутка |
| 1.3.2 | Бестселлеры: карусель или swipe | Карточки не сжимать; карусель/слайдер или 2 колонки |
| 1.3.3 | Общий аудит мобильной версии главной | Проверить читаемость текста, размер тапов, перекрытия |

### 1.4 Instagram — «Obserwuj nas na Instagramie»
| ID | Задача | Детали |
|----|--------|--------|
| 1.4.1 | Убрать три одинаковые кнопки «Zobacz post» | Не дублировать одну и ту же кнопку |
| 1.4.2 | Встроить реальные превью постов | Свежие изображения из ленты (API/виджет Instagram или ручной блок) |
| 1.4.3 | Заменить текст на дату/короткое описание | Вместо одного текста — дата или краткий текст под каждым превью |
| 1.4.4 | Согласовать способ получения постов | Ограничения Meta API; при необходимости — статичные блоки с ручным обновлением |

### 1.5 Cookie-баннер (GDPR)
| ID | Задача | Детали |
|----|--------|--------|
| 1.5.1 | Баннер виден в графическом интерфейсе | Проверить z-index, отображение в браузере (не только текстовая версия) |
| 1.5.2 | Не перекрывать важные элементы | Позиция (низ/верх), отступы, не перекрывать кнопки |
| 1.5.3 | Кнопка «Ustawienia» | Открывает настройки согласия (категории cookie, отказ/согласие) |
| 1.5.4 | Соответствие GDPR | Хранение выбора, возможность отозвать, ссылка на политику |
