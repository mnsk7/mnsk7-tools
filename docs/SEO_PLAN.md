# SEO Plan — mnsk7-tools.pl

*(Выход агента 02_growth_seo)*

Структура категорий, лендинги под запросы, шаблоны мета, schema. Источники: REQUIREMENTS, DISCOVERY, CLIENT_INTERVIEW_SUMMARY, AS_IS_AUDIT.

---

## 1. Цели SEO

- Органический трафик по запросам: **фрезы для алюминия**, **фрезы для MDF**, **фрезы для дерева**, **CNC фрезы** и смежные (тип инструмента + материал/применение).
- Корректная индексация каталога и карточек; качественные сниппеты (Title, Description, цена, наличие).
- Внутренняя перелинковка по типу инструмента и материалу обработки.
- Раздел статей/инструкций — долгий контент и экспертность (REQUIREMENTS 3.1).

---

## 2. Структура категорий и подкатегорий

Логика «как люди ищут» (DISCOVERY): сначала **тип инструмента**, затем параметры.

**Рекомендуемая структура (без жёсткой привязки к текущему дереву):**

- По **типу инструмента:** концевая, шаровая, гравировальная, для планирования, со сменными пластинами, с подшипником и т.д.
- По **материалу обработки:** дерево, MDF, алюминий, сталь, пластик, композиты — как подкатегории или фильтры + отдельные лендинги.
- Не смешивать в одной ветке тип, материал и «технические параметры» как категории (текущая боль из интервью).

**Связь с каталогом:** см. задачи E2 в 010_epics.md и S2-01, S2-02 в 020_sprint_01 / 030_sprint_02.

## 3. Страницы-лендинги под запросы

Целевые страницы (уникальный контент, не только список товаров):

| Запрос / тема | Тип страницы | Примечание |
|---------------|--------------|------------|
| Фрезы для алюминия | Лендинг категории или отдельная страница | Текст 300–800 слов + товары (фильтр по материалу) |
| Фрезы для MDF | То же | + внутренние ссылки на «фрезы для дерева», «CNC фрезы» |
| Фрезы для дерева | То же | Хиты: фрезы для планирования, с подшипником (DISCOVERY) |
| CNC фрезы | Обзорная страница или категория | Ссылки на материалы и типы |
| Фрезы со сменными пластинами | Категория + текст | Самая маржинальная категория (DISCOVERY) |

Каждая такая страница: уникальные Title, H1, Meta Description, блок текста (как выбрать, чем отличаются), при необходимости FAQ (см. CONTENT_PLAN).

---

## 4. Шаблоны мета (Title / H1 / Description)

**Категория (пример):**
- **Title:** `{Название категории} do {материал/операция} | MNK7 Tools`  
  Пример: `Frezowanie drewna i MDF | MNK7 Tools`
- **H1:** совпадает с логикой страницы (название категории или ключевой запрос).
- **Description:** 2–3 предложения (что за инструмент, для чего) + USP (доставка на следующий день, ассортимент) + призыв. 150–160 символов.

**Карточка товара:**
- **Title:** `{Тип} {Ø} {покрытие/материал} do {материал обработки} | MNK7 Tools`  
  Пример: `Fresa walcowa 6mm do aluminium | MNK7 Tools`
- **H1:** название товара (с ключевыми параметрами при необходимости).
- **Description:** краткое описание + ключевые параметры + наличие/доставка. Уникально для каждого товара.

**Обязательное (seo_woocommerce):**
- ЧПУ (permalinks) включены; canonical на категориях и товарах.
- Уникальные Title/H1 на всех индексируемых страницах.

---

## 5. Индексация фильтров

- **Правило:** не индексировать тонкие комбинации фильтров (много URL с малым контентом). Выбранный фильтр-плагин должен уметь noindex для параметрных URL или отдавать canonical на страницу категории.
- **robots.txt:** текущий `Disallow: /?` блокирует все GET-параметры (AS_IS_AUDIT). После смены фильтра пересмотреть: разрешить нужные параметры для фильтрации без индексации мусорных комбинаций (noindex в meta или через плагин).
- **Sitemap:** категории и товары — в sitemap; отфильтрованные URL (если не noindex) — по решению (часто исключают).

---

## 6. Хлебные крошки и Schema

- **BreadcrumbList:** на всех страницах каталога и карточках (Yoast или выбранный плагин).
- **Product schema:** в карточке товара — price, availability, sku, name, image (Yoast уже даёт; убрать дубль от schema-and-structured-data-for-wp — AS_IS_BACKLOG P2-01).
- **FAQ schema:** на страницах с FAQ-блоком (категории, лендинги) — см. CONTENT_PLAN.
- **Organization / LocalBusiness:** на главной или в футере при необходимости.

---

## 7. Внутренняя перелинковка

- На страницах категорий: ссылки на «фрезы по материалам», «по типу операции», смежные категории.
- В текстах статей (раздел блога/инструкций): ссылки на категории и хитовые товары.
- В карточке: «похожие», «часто покупают с этим» — дополнительная перелинковка.
