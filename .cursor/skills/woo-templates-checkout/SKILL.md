# Woo templates + checkout

## UX чекаута

- минимум полей
- ясные ошибки
- доставка/оплата понятны
- подтверждение заказа с понятным next step

## Технически

- любые изменения через hooks/filters
- кастом-поля хранить корректно (order meta)
