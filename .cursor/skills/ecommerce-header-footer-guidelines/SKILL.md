# Header и footer — что должно быть на сайте

**Засада:** Приоритет — **удобство пользователя и стандарты e-commerce**. Пожелания учитываем только если не ухудшают UX/конверсию.

## Header

- Главный ряд: logo (домой), поиск, аккаунт/корзина.
- Меню: **Sklep (каталог)** + O nas + Pomoc + Kontakt.

## Footer

- Контакт: email/тел/часы.
- Instagram: 1 ряд постов + ссылка на профиль.
- Ссылки: Pomoc, Dostawa, Regulamin, Polityka, Kontakt.

## Mobile

- hamburger menu, поиск и корзина доступны сразу
