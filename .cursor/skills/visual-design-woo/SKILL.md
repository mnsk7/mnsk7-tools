# Visual design — sklep Woo (B2B narzędzia)

## Cel

Сайт должен выглядеть как **профессиональный магазин**: порядок, читаемость, доверие.

## Zasady

- **Сплочённость:** один набор цветов, одна базовая типографика.
- **Воздух:** не уплотнять; отступы помогают сканировать.
- **Иерархия:** цена/CTA/наличие — главные; вторичное — меньше.
- **Кнопки:** один главный стиль для “Добавить в корзину”.
- **Доверие:** доставка/оплата/“Faktura VAT” рядом с CTA или внизу.
