# QA Smoke Woo

## Цель

Быстро подтвердить, что после изменений не сломан конверсионный флоу WooCommerce.

## Минимум (обязательное)

- add to cart из категории и из карточки (P0)
- корзина: товар виден, qty меняется (P0)
- checkout entry: страница открывается, форма видима (P0)

## Если в проекте используется

- купон/скидка (если есть)
- checkout end-to-end (заказ создаётся)
- письмо на staging перехвачено staging-safety (не уходит наружу)
- статус заказа корректный (pending/processing)
