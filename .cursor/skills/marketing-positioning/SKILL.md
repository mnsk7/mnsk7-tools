# Marketing positioning

## Формула оффера для фрез

- Для кого (цех/мастерская)
- Для чего (операция/материал)
- Почему мы (наличие/подбор/качество/доставка)
- Доказательства (отзывы, спецификации, гарантия)

## Блоки, которые поднимают конверсию

- "Подбор по материалу" (быстрый вход)
- "В наличии / отправка сегодня" (если правда)
- "Совместимость" (цанга/хвостовик)
- "Рекомендации"
- "Возврат/гарантия" рядом с CTA
