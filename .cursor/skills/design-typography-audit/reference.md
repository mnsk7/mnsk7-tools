# Справка: шаблон отчёта и чек-листы

## Шаблон отчёта аудита

```markdown
# Аудит дизайна и типографики — [дата]

## 1. Типографика (шрифты)

### В коде
- Единый шрифт: да/нет. Источник: `--font-base` в `01-tokens.css`.
- Найденные вхождения `font-family` вне токена: [файл:селектор — значение].
- Размеры шрифтов: везде ли `--fs-*` или есть жёсткие px/rem.

### Рекомендации
- [ ] Заменить все font-family на var(--font-base).
- [ ] Заменить жёсткие размеры на --fs-xs … --fs-hero.

---

## 2. Отступы и обводки

### В коде
- Отступы (margin/padding/gap): перечень селекторов с «магическими» значениями вместо --space-*.
- Границы и радиусы: перечень с жёсткими значениями вместо --color-border*, --r-sm/md/lg.

### Однотипные блоки (консистентность)
| Тип блока   | padding      | margin       | border-radius |
|------------|--------------|--------------|---------------|
| Карточка   | …            | …            | …             |
| Кнопка     | …            | …            | …             |
| Секция     | …            | …            | …             |

### Рекомендации
- [ ] Вынести повторяющиеся значения в токены --space-* / --r-*.
- [ ] Унифицировать отступы у однотипных блоков.

---

## 3. Код vs реальность (если проверялось в браузере)

| Элемент (страница) | Свойство     | Ожидалось (токен) | Фактически (Computed) | Примечание |
|--------------------|-------------|-------------------|------------------------|------------|
| …                  | font-family | Inter             | …                      | …          |
| …                  | padding     | 16px (--space-16) | …                      | …          |

Расхождения: [кратко — какой селектор перебивает].
```

## Чек-лист быстрой проверки (код)

- [ ] В теме нет второго `@font-face` кроме Inter.
- [ ] Нет `font-family:` с литералом, отличным от использования `var(--font-base)`.
- [ ] Нет `font-size:` с px/rem, которые дублируют значения --fs-*.
- [ ] Повторяющиеся margin/padding заданы через --space-*.
- [ ] border-radius заданы через --r-sm/md/lg.
- [ ] Цвета границ — через --color-border / --color-border-strong.
